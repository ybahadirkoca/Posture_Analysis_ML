# Posture Analysis with IMU and Machine Learning

This repository contains feature matrices and code for the study:

**"Anatomy-Based Assessment of Spinal Posture Using IMU Sensors and Machine Learning"**

## Contents
- `/data`: Derived feature matrices (angular deviations, demographic & lifestyle data).  
  *(Raw IMU recordings are not shared due to ethical restrictions.)*  
- `/code`: Python scripts for preprocessing, feature extraction, data balancing, and ML evaluation.  
- `/docs`: Supplementary materials (figures, plots, etc.).

## Requirements
- Python 3.9+
- scikit-learn
- pandas
- numpy
- matplotlib
